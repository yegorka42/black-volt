# Black-Volt - Сайт электромонтажной компании

## Описание проекта

Black-Volt - это современный, стильный и функциональный сайт для электромонтажной компании, выполненный в фирменном стиле с черно-оранжевой цветовой гаммой. Сайт полностью адаптивный и включает все необходимые разделы для представления услуг компании.

## Основные характеристики

### Дизайн
- **Цветовая схема**: Черный (#000000) и оранжевый (#ff6b35)
- **Фирменный стиль**: Современный, технологичный, профессиональный
- **Типографика**: Шрифт Montserrat с различными весами
- **Иконки**: Font Awesome для визуального оформления

### Разделы сайта
1. **Header** - Шапка сайта с навигацией и контактным телефоном
2. **Hero** - Главный экран с призывом к действию
3. **Services** - Блок услуг компании (4 типа услуг)
4. **About** - Информация о компании и преимуществах
5. **Contacts** - Контактная информация и форма обратной связи
6. **Footer** - Подвал сайта с дополнительной информацией

### Функциональность
- **Адаптивный дизайн** - Корректное отображение на всех устройствах
- **Мобильное меню** - Бургер-меню для мобильных устройств
- **Плавная прокрутка** - Навигация по якорям с плавной анимацией
- **Валидация форм** - Проверка данных в форме обратной связи
- **Анимации** - Интерактивные элементы и анимации при скролле
- **Кросс-браузерность** - Поддержка всех современных браузеров
